查找浮点数的平方根

不能用传统的闭区间写法，因为整数是离散的，±1不会影响，但浮点数的话跨越太大了，会跳过很多
```cpp
double mySqrt(double x)
{
    double l = 0;
    double r = max(1.0, x); 
    // r=max(1.0,x)，因为小数情况下根本不清楚最大边界到底是多少好，但一定小于1
    double mid;

    while (r - l > 1e-10) // 这是科学计数法 ，1×10^(-10)
    {
        mid = l + (r - l) / 2;
        if (mid <= x / mid) 
        {
            l = mid;
        }
        else
        {
            r = mid;
        }
        // 采用 <=/> 或 >/<= 都没区别，因为精度足够了
    }
    return l + (r - l) / 2; // 这里返回l / r / l+(r-l)/2都可以，精度足够了
    // l 永远满足 l² <= x
    // r 永远满足 r² > x
    }
```
