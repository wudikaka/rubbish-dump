给你一个非负整数 x ，计算并返回 x 的算术平方根 
由于返回类型是整数，结果只保留整数部分 ，小数部分将被舍去


核心思想：牛顿迭代法(对于目前阶段的我来说只用来求平方根即可 26.2.10)
求解方程：f(x) = x² - a = 0 的根（即 √a）
>先随便猜一个初始值 x₀，
>在点 (x₀, f(x₀)) 处画切线，
>切线交x轴的点 x₁ 会比 x₀ 更接近真实根，
>重复这个过程，越来越接近真实根。

>导数 f'(x) = 2x，
>点 (xₙ, f(xₙ)) 的切线方程：y - f(xₙ) = f'(xₙ)(x - xₙ)，
>切线与x轴交点xₙ₊₁ = (xₙ + a/xₙ) / 2

>说白了就是猜个值，不停x = (x + a/x) / 2
>然后就趋近于真实值
```cpp
class Solution
{
public:
    int mySqrt(int x)
    {
        if (x <2)
            return x;
        long long guess = x;//首次猜测值

        while (guess * guess > x)//当guess逼进真实值，会出现小数，但cpp会把小数部分忽略，
                                 //此时guess*guess就是x，达到条件，终止
        {
            guess = (guess + x / guess) / 2;
        }
        return (int)guess;
    }
};
```
