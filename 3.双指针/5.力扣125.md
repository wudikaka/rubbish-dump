如果在将所有大写字符转换为小写字符、并移除所有非字母/数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个回文串  
给个字符串 s，如果是回文串，返回 true ；否则，返回 false   

示例：  
输入: s = "A man, a plan, a canal: Panama"  
输出：true  
解释："amanaplanacanalpanama" 是回文串  

我的方法：  
用双指针指向每个位置，遇到非字母数字的元素就跳过，是大写就用ASCII值改为小写，然后比较。  
错因：只用if进行一次检查，忽略了可能下一位也不是字母数字  
***结论：库函数isalnum(a)可以判断a是否为字符或者数字***  

```cpp
class Solution
{
public:
    bool isPalindrome(string s)
    {
        int l = 0;
        int r = s.size() - 1;
        while (l < r)
        {
            while (!isalnum(s[l]) && l < r)
            {
                ++l;
            }
            while (!isalnum(s[r]) && l < r)
            {
                --r;
            }
            
            if (s[l] < 'a')
            {
                s[l] += 32;
            }
            if (s[r] < 'a')
            {
                s[r] += 32;
            }

            if (s[r] != s[l])
            {
                return false;
            }
            ++l;
            --r;
        }
        return true;
    }
};
```

遍历方法:  
先遍历把数字/字母提前到sgood中，然后创建新的倒过来的字符串，进行比较，若是回文，倒过来也一样
isalnum(ch)判断是否为数字或字母
tolower(ch)把大写转为小写

```cpp
class Solution 
{
public:
    bool isPalindrome(string s) 
    {
        string sgood;
        for (char ch: s) 
        {
            if (isalnum(ch)) 
            {
                sgood += tolower(ch);
            }
        }
        string sgood_rev(sgood.rbegin(), sgood.rend());
        //构造个sgood_rev字符串，初始化时用sgood倒着初始化
        return sgood == sgood_rev;
    }
};
```
